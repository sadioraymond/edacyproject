Application de gestion scolaire
L'application permet à optimiser tout le parcours scolaire

Fonctionnalités Développées:
Ajout Séries
Lister Série
Modification Série
Suppression Série
Ajout Classe
Modification info Classe
Suppression Classe
Inscription éléves( Ajout élèves et ajout inscription)
Lister éléves
Modifier info éléves
Supprimer éléves
Liste des inscriptions
Modifier Inscriptions
Supprimer Inscription
Ajout Année Académique
Lister Année Académique
Cloturer Année Académique
Modifier Année Académique
Supprimer Année Académique

Fonctionnalités Restantes

Lister classe par série
Lister éléves par classe et année académique
Lister classe par année académique
Lister éléves par Année Académique
Contrôle ancien éléves pendant l'iscription
Contrôle des attributs à la saisie
Réajustement du template(Redéfinition des icones pour les onglets et affichage info user)

Pour accédrer a l'application il faut se connecter avec un login et un mdp

login:demba@sarr.com
mdp:1234

Configuration:

java version: 1.8

Spring boot framework 1.5.10

Eclipse Java EE IDE for Web Developers.
Version: Oxygen.2 Release (4.7.2)
Build id: 20171218-0600

Base de donnée Mysql obligatoire
Configuration de connexion à la base dans applications.properties (src\main\resources)

Build and Run:

Importer le projet en tant que projet maven.

Run en tant que java application


NB:

-Pour les fonctionnalités de modification, suppression et cloture c'est au niveau des icones sur les différentes listes
